# Scalar and Elementwise Matrix Algebra (uint64)

Exercises trace, scalar multiplication, elementwise addition, and the Hadamard product on `uint64` matrices, asserting distributivity and trace linearity.

Validate and run from this directory:

```sh
stoffel check .
stoffel run .
```

The program asserts its own results and prints a summary; a non-zero exit
or a failed assertion means the example regressed.
