# Euclidean Algorithm, Iterative (int64)

The classic remainder-based Euclidean algorithm as an `int64` loop, normalised for negative inputs, validated on coprime, nested, and equal-value pairs along with gcd identities.

Validate and run from this directory:

```sh
stoffel check .
stoffel run .
```

The program asserts its own results and prints a summary; a non-zero exit
or a failed assertion means the example regressed.
