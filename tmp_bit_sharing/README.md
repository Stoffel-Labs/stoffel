# Stoffel Project

Validate the Stoffel source:

```sh
stoffel check
```

Run the default local MPC example:

```sh
stoffel run --input a=40 --input b=2
```

Run the default example once with the development command:

```sh
stoffel dev --once --input a=40 --input b=2
```

Build bytecode:

```sh
stoffel build
```
