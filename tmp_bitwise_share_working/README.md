# Stoffel Project

Validate the Stoffel source:

```sh
stoffel check
```

Run the default local MPC example:

```sh
stoffel run
```

Run the default example once with the development command:

```sh
stoffel dev --once
```

Build bytecode:

```sh
stoffel build
```

Build the Rust SDK wrapper with included bindings:

```sh
cargo build
```

Run the Rust SDK wrapper:

```sh
cargo run
```
